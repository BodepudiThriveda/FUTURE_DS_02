📌 Project Overview

I created an interactive Customer Churn Analysis Dashboard using Tableau that helps businesses understand patterns and drivers of customer churn.

This dashboard highlights key trends and insights that help identify which customer segments are most at risk of churning and what factors are influencing churn behavior. It enables data-driven strategies to improve retention and overall business growth.

🔹 Key Features

Interactive Filters: Explore churn patterns by customer segments, demographics, and time periods

Churn Trend Visualizations: Track churn over time and across different groups

Insightful Metrics: Understand drivers of churn and performance impact

Data visualization and storytelling using Tableau

Dashboard design for real-world business insights

Interpreting patterns and translating data into actionable insights
